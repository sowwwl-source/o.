O.

Le réseau minimal.

// manif3st // // qu3st // // login // // état //

Manifeste O. O. n’est n i site. O. est n i état. Un point posé après la phrase, quand tout le reste a été dit. Un arrêt volontaire dans le flux. Une résistance minimale.

O. refuse l’accumulation. Ce qui n’est pas nécessaire est retiré. Ce qui reste devient lisible.

Ici, la stabilité n’est pas un point de départ, c’est une conquête. Le calme n’est pas promis, il est évalué. L’équilibre ne se proclame pas : il se prouve.

O. observe sans exploiter. Il ne capte pas pour vendre, il capte pour réguler. L’algorithme n’est pas un piège, c’est un miroir. Il ne vous pousse pas, il vous replace.

Clique. Inverse. Recommence.
