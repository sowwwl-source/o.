- 01_ARCHITECTURE: architecture 5 strates, schema Live 3.0
- 02_PATCH_RULES: regles de patch

- 04_INTERFACES: interfaces humaines
- 05_MACHINES: machines principales

- 09_AUDIO_ROUTING: routage audio
- 10_ADMIN: budget, planning, checklist
